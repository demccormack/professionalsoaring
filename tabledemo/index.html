Tables Demo

Putting my CSS in a separate file means I can use the same styles on every page of my website. The background image and shaded container on this page are showing here because I inserted a 'link' tag in the 'head' of this html page, to the CSS file I created for the other pages on my website.

The table styles are in another CSS file, tabledemo.css, which I created just for this demo, and again linked in the 'head'. By assigning both tables to the 'bitwise' class, I can ensure that all my tables related to bitwise operations will have the same style. Any changes I make to the CSS will propagate to every table in the class.

This can be done even more simply without defining a class and instead creating CSS styles for 'table', 'td' and 'th'. The drawback is that I wouldn't be able to define a second style of table.
